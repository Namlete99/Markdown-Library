# Cấu trúc thư mục - Folder Structure

## Cài đặt và sử dụng

1. Cài đặt tiện ích mở rộng (extension) thông qua Visual Studio Code (Xem ở [1]ở **tài liệu tham khảo**).
2. Nhấp chuột phải vào thư mục mong muốn trong trình quản lý tệp VS Code.
3. Chọn **Generate Markdown Structure** từ menu ngữ cảnh.
4. Cấu trúc thư mục được định dạng bằng Markdown sẽ tự động được sao chép vào Clipboard của bạn.
5. Hãy dán đoạn mã này vào nơi khác của tệp hoặc tài liệu mà bạn cần.
