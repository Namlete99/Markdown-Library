# Cấu trúc thư mục - Folder Structure

## Cài đặt và sử dụng

1. Cài đặt tiện ích mở rộng (extension) thông qua [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=jmkrivocapich.drawfolderstructure#:~:text=Flexible%20Exclusions:%20Exclude%20specific%20files,js%22).
2. Nhấp chuột phải vào thư mục người dùng mong muốn trong trình quản lý tệp Visual Studio Code.
3. Chọn **Generate Markdown Structure** từ menu ngữ cảnh.
4. Cấu trúc thư mục được định dạng bằng Markdown sẽ tự động được sao chép vào Clipboard của bạn.
5. Hãy dán đoạn mã này vào nơi khác của tệp hoặc tài liệu mà bạn cần.

Ví dụ:

````markdown
```
└── 📁E.1. Tài liệu
    ├── 1. Mục lục.md
    ├── 2. Cấu trúc thư mục.md
    ├── 3. Văn bản thuần.md
    ├── 4. Các phàn tử khối.md
    ├── 5. Đặc biệt.md
    ├── 6. Biểu thức toán học.md
    └── references.bib
```
````

Kết quả:

