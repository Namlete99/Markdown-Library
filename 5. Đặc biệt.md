# 1. Liên kết - Link

Chèn trực tiếp hoặc gián tiếp link:

- Trực tiếp thì cứ dán (paste) link vào là xong

Ví dụ: Link nhạc cổ động câu lạc bộ Manchester United: https://youtu.be/611WYDonzTU?si=qZADiVphkX2KD_LE

- Gián tiếp sử dụng cú pháp:

```markdown
[text]{link}
```

Ví dụ:

```markdown
Nhạc cổ động Manchester United, nhấn [tại đây](https://youtu.be/611WYDonzTU?si=qZADiVphkX2KD_LE) để nghe
```

Kết quả:

Nhạc cổ động Manchester United, nhấn [tại đây](https://youtu.be/611WYDonzTU?si=qZADiVphkX2KD_LE) để nghe

---

# 2. Hình ảnh - Image

Chèn trực tiếp hoặc gián tiếp hình ảnh:

- Trực tiếp thì cứ dán (paste) link vào là xong

Ví dụ: Link ảnh Glory Glory Glory Man United

https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcSAJNiqjBRK7IsT-hNyYaSC5PukVPzkSSHJCw&s

- Gián tiếp sử dụng cú pháp sau:

```markdown
![alt](Link ảnh)
```

Ví dụ:

```markdown
![Đội hình quỷ đỏ C1 mùa ko nhớ](https://cly.1cdn.vn/2024/02/16/mun.jpg)
```

Kết quả

![Đội hình quỷ đỏ C1 mùa ko nhớ](https://cly.1cdn.vn/2024/02/16/mun.jpg)

---

Ta có thể canh giữa ảnh, cũng như là chú thích cho ảnh bằng cách sử dụng cặp thẻ `<div>...</div>` với thuộc tính bên trong là `align = "center"`

Ví dụ:

```markdown
<div align="center">

![](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRZfLuviePOqQJ1yuhvxr80AsqV91Btr2rqZw&s)

Logo Github lụm được được trên Internet

</div>
```

Kết quả:

<div align="center">

![](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRZfLuviePOqQJ1yuhvxr80AsqV91Btr2rqZw&s)

Logo Github lụm được được trên Internet

</div>

# 3. Đường kẻ ngang - Horizonal rules

Để tạo đường kẻ ngang, sử dụng 3 dấu `*` hoặc `-` hoặc `_` trên một dòng.

Cú pháp:

```markdown
---
***
___
```

Kết quả (đều như nhau):

---

---

---

# 4. Escape markdown

Đôi khi trong lúc soạn văn bản sẽ cần những kí hiệu trùng với cú pháp của markdown. Để phân biệt, ta chỉ cần thêm dấu `\` trước những kí hiệu đó là được.

Cú pháp:

```markdown
\`hai dấu nháy\`

\*\*\*ba dấu sao hai bên\*\*\*
```

Kết quả:

\`hai dấu nháy\`

\*\*\*ba dấu sao hai bên\*\*\*
